How to Compress a Video to an Exact File Size

Updated July 2026 · 8 min read · By VidZip Team

Most video compressors make you guess. Pick a quality preset, hit Compress, hope the result fits under whatever limit you're trying to hit. Target size mode flips that around: you tell the compressor the exact file size you need, and it does the math to hit it. This guide explains how target size compression works, when to use it, and how to nail any size limit — 25 MB for email, 16 MB for WhatsApp, 100 MB for upload — every time.

The Problem With Quality Presets

Traditional compressors (and Apple's built-in tools) work in quality terms: pick "High," "Medium," or "Low," or pick a resolution, and the app gives you whatever file size comes out. That works fine when you just want to free up storage. But it's useless when you have a hard size limit.

Consider the common case: you need to email a video, and Gmail caps attachments at 25 MB. With quality presets, you compress, check the result, and if it's over 25 MB, you try a lower preset — repeat until it fits. Slow, frustrating, and the result is usually either over or way under the limit.

Target size mode solves this: type "25 MB," compress once, done.

How Target Size Compression Works

The math is simple. File size is essentially bitrate × duration, so to hit a target size, you reverse the formula:

required bitrate = (target size in bits) ÷ (duration in seconds)

remember: 1 byte = 8 bits, so 25 MB = 200 megabits

Worked example — you want a 60-second video to fit under 25 MB:

  • 25 MB × 8 = 200 megabits
  • 200 Mb ÷ 60 seconds ≈ 3.3 Mbps

Set the encoder to an average bitrate of 3.3 Mbps, run it for 60 seconds, and the file lands at roughly 25 MB. VidZip does this calculation automatically — you just enter the target.

When Target Size Doesn't Make Sense

Target size compression trades quality for predictability. It's perfect when you have a hard size cap. But there are cases where it's the wrong tool:

  • Long videos at small targets. A 10-minute video compressed to 25 MB will look blocky because the bitrate is too low. Use a cloud link instead.
  • Quality-first compression. If you're archiving footage and care about quality, use a preset (High Quality) instead of a target.
  • Batch compression. Target size is per-video. For batches, use a preset so every clip gets the same quality profile.

For everything else — hitting email/messaging/upload limits — target size mode is exactly the right tool.

Frequently Asked Questions

How do I compress a video to a specific file size?

Use a compressor that supports target size mode, like VidZip. Enter your target (25 MB, 16 MB, etc.) and the app calculates the bitrate needed: target bits ÷ duration = required bitrate. The result hits the target size automatically.

How do you calculate bitrate to hit a target file size?

Bitrate = (target size in bits) ÷ (duration in seconds). For a 25 MB target on a 60-second video: (25 × 8,000,000) ÷ 60 ≈ 3.3 Mbps. Set the encoder to that average bitrate and the file lands at the target.

What video size fits in an email?

Gmail, Outlook, and most email providers cap attachments at 25 MB. Set your target to 25 MB (or 20 MB to be safe) and the compressed video will send as an attachment without being rejected.

Does target size compression hurt quality?

It depends on the target relative to the video's duration. A 25 MB target on a 30-second clip looks near-original. The same 25 MB on a 10-minute video will be noticeably degraded because the bitrate is spread thin. VidZip shows a quality estimate before you compress so you know what to expect.

Can I hit an exact file size in iMovie or Photos?

No. iMovie and Apple Photos only let you choose resolution presets (4K, 1080p, 720p), not a target file size. To hit an exact MB target, you need a compressor with target size mode like VidZip.
